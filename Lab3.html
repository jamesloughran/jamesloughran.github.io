Bootstrap is a free open-source front-end framework that styles and makes your webpages responsive. Created by Mark Otto and Jacob Thornton from twitter, it quickly became a great mobile-first responsive framework. Since 2011 it has been open-source on GitHub and used to this day by many developers.
